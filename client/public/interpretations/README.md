# Interpretation Content

Edit these Markdown files to change the texts shown after each numerology calculation.

Each interpretation lives in its own file so you can write freely without touching React or JavaScript syntax.

Folders:

- name/yo-exterior/1.md to 9.md: interpretations for consonant-derived numbers.
- name/yo-interior/1.md to 9.md: interpretations for vowel-derived numbers.
- name/yo-real/1.md to 9.md: interpretations for full-name numbers.
- birthdate/yo-espiritual/1.md to 9.md: interpretations for birthdate-derived numbers.

You can write regular paragraphs. Blank lines are preserved. Markdown syntax is optional; the app currently renders the content as readable plain text.
